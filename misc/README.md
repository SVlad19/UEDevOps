# Unreal Engine Utility Scripts

All files from this folder are copied to `devops_data` after running `setup.bat`.

## Scripts

* `clean_intermediate_files.bat` — removes temporary Unreal Engine files. The cleanup list can be changed in `devops_data/config.bat`.
* `format_all_files.bat` — formats source files using `.clang-format`.
* `generate_project_files.bat` — generates Visual Studio solution files.
