# Unreal Engine Cooking Scripts

This folder contains scripts for cooking Unreal Engine project content for different targets.

## Scripts

* `cook_game.bat` — cooks content for the Game target.
* `cook_client.bat` — cooks content for the Client target.
* `cook_server.bat` — cooks content for the Server target.

## Requirements

Client and Server cooking require an Unreal Engine build compiled from source.

Before running the scripts, make sure the project and engine paths are correctly configured in `config.bat`.
