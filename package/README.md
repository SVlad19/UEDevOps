# Unreal Engine Packaging Scripts

This folder contains scripts for packaging Unreal Engine projects.

## Scripts

* `package_game.bat` — packages the Game target.
* `package_client.bat` — packages the Client target.
* `package_server.bat` — packages the Server target.
* `package_client_with_server.bat` — packages both Client and Server targets.

Client and Server packaging require an Unreal Engine build compiled from source.
